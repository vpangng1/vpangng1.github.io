The Travel App

Building intuitive app interfaces using digital design toolkits.

ABOUT THE TRAVEL APP

The Travel App was part of a side project I created to practice building UI designs in Sketch. After creating the prototypes, I was able to upload them into InVision using the Craft plugin. In InVision, I was able to experiment with some interactive elements and see how they would look like on different mobile devices.

I started this project using Sketch to create the interface and added the content. I then used free photos from Pexel Images to fill in some of the images for the screen interface. I used vectors to create the download and bookmark icons. Lastly, I used the Sketch mockup template that I downloaded from Dribble to insert the screen into the iPhone GUI.

After completing the screen interface, I used the Craft plugin in Sketch to upload them to InVision, where I was able to do further prototyping.
