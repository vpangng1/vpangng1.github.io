Travel App

Foray into building mobile app interfaces and UI design.

ABOUT TRAVEL APP

The Travel App was part of a side project I created to practice building UI designs in Sketch. After creating the prototypes, I was able to upload them into InVision using the Craft plugin. I was then able to experiment with some interactive elements and see how they would look like on different mobile devices.

I started this project using the Craft freehand tool to create the mockups. I then translated those designs into Sketch and added the content. I used a combination of free photos from Pexel Images and the data panel from Craft to randomly fill the image placeholders.

I added the airplane icon for the loading screen from a Material Design Icons Sketch file. After completing the screen interface, I used the Craft plugin in Sketch to upload them to InVision, where I was able to see how the screen mockups would look like on different mobile devices.
